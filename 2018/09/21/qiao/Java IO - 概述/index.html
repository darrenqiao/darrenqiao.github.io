Java IO - 概述

发表于 2018-09-21 | 更新于 2018-09-27 | 分类于 Java | 评论数： | 阅读次数：

Java IO是由JavaSE提供的API接口，主要目的是用于读写数据。功能主要集中于 java.io(同步阻塞实现)，java.nio(异步阻塞实现)和java.nio2(异步非阻塞实现)三个包中,这个系列主要讲java.io。

首先，IO处理涉及到一下几个基本概念，

数据来源

  • 文件
  • 管道
  • 网络连接
  • 内存缓存
  • Java标准输入输出,错误输出

数据传输的格式

  • 流，是Java IO中数据存在和传输的格式，表示一组有序的，有起点，有重点的集合。

数据怎么读

  • 字节读取
  • 字符读取
